 36class BaseNetModel:
 37    """
 38    The BaseNetModel class provides a wrapper for the tf.keras.model API with easier use. When initialized,
 39    it initializes a breech of databases in its attribute 'breech'. If we provide a compiler, the model will be
 40    built from the compiler; however, if we provide a tf.keras.model, the compiler is ignored and the model is built
 41    from the provided tf.keras.model.
 42
 43    To add a database to the model, we can use the method BaseNetModel.add_database() that takes a BaseNetDatabase as
 44    input.
 45
 46    The class contains load and save methods to store the compilers (.cpl files) and models (.h5 files) in the same
 47    directory.
 48
 49    We also provide a BaseNetModel.fit() method that can create a separate process for training. The original framework
 50    does not include this feature:
 51        *   The BaseNetModel.fit() method takes as input the index of the loaded database via
 52        BaseNetModel.add_database() method and takes the train and validation subsets to fit the model.
 53        *   If the training process should not block the main process, the parameters 'avoid_lock' must be set to True,
 54        in that case, another process will take over the fitting tf.keras.model.fit() method and the information will
 55        be updated in the return class: BaseNetResults.
 56        *   In case we avoid the main process to be locked with the 'avoid_lock' feature, we will need to recover the
 57        tf.keras.model with the BaseNetModel.recover() method once the training is finished (check
 58        BaseNetResults.is_training).
 59
 60    We can also evaluate the performance of the database with the BaseNetModel.evaluate() method, that makes use of the
 61    test subset.
 62
 63    We can also predict the output of a certain input with the BaseNetModel.predict() method.
 64
 65    We can also visualize the model with the BaseNetModel.print() method in a PNG image.
 66    """

 68        """
 69        The BaseNetModel implements an API that makes use of keras and tensorflow to build Deep Learning Models.
 70        :param compiler: BaseNetCompiler object to build the model.
 71        :param model: If a keras.model is already compiled, you can import it in the model parameter, so the compiler
 72        won't be used during the construction.
 73        """
 74        self._verbose = verbose

 76        self.compiler = compiler
 77        self.is_trained = False
 78        self.name = name
 79        self.breech = []

 81        try:
 82            if model is not None:
 83                self.model = model
 84                if not self.name:
 85                    self.name = 'unnamed_model'
 86                self.is_compiled = True
 87                self.summary = model.summary()

 89            elif self.compiler is not None:
 90                self.model = self._build()
 91                if not self.name:
 92                    self.name = compiler.name
 93                self.is_compiled = True
 94                self.summary = self._get_summary()

 97                self.model = None
 98                self.is_compiled = False
 99                if not self.name:
100                    self.name = 'uncompleted_model'
101                self.summary = 'uncompleted_model'
102
103        except Exception as ex:
104            if not self.name:
105                self.name = 'uncompleted_model'
106            self.model = None
107            self.is_compiled = False
108            self.summary = ex
109            logging.error(f'BaseNetModel: The model is empty. Raised the following exception: {ex}.')

111    def fit(self, ndb, epochs, tensorboard: bool = True, avoid_lock: bool = False):
112        """
113        This function fits the BaseNetModel with the selected database.
114        :param ndb: Index of the database already loaded.
115        :param epochs: Number of epochs to train.
116        :param tensorboard: Activates or deactivates the Tensorboard.
117        :param avoid_lock: Avoids the training process to lock the parent process.
118        :return: BaseNetResults of the fitting process.
119        """
120        if tensorboard:
121            self._flush()
122            tb = program.TensorBoard()
123            tb.configure(argv=[None, '--logdir', f'{__tensorboard_logs__}/{self.name}'])
124            url = tb.launch()
125            webbrowser.open(url, new=2)
126
127        if ndb < len(self.breech):
128            db = self.breech[ndb]
129        else:
130            if self._verbose:
131                logging.warning('BaseNetModel: Cannot load the BaseNetDatabase to fit, the index of the '
132                                'database does not exist.')
133            return None
134
135        xtrain = db.xtrain
136        ytrain = db.ytrain
137        xval = db.xval
138        yval = db.yval
139
140        __history__ = None
141        fit_callback = _FitCallback()
142        try:
143            if avoid_lock:
144                keras.models.save_model(self.model, __bypass_path__)
145                queue = Queue()
146                p = Process(target=self._fit_in_other_process, args=((xtrain, ytrain), (xval, yval),
147                                                                     epochs, db.batch_size, self.name, queue, db.dtype))
148                p.start()
149                __history__ = BaseNetResults(queue=queue, parent=p)
150            else:
151
152                # Auto shard options. Avoid console-vomiting in TF 2.0.
153                options = tf.data.Options()
154                options.experimental_distribute.auto_shard_policy = tf.data.experimental.AutoShardPolicy.OFF
155                # Re-formatting the database.
156                _xtrain = tf.convert_to_tensor(xtrain, dtype=getattr(tf, db.dtype))
157                _ytrain = tf.convert_to_tensor(ytrain, dtype=getattr(tf, db.dtype))
158                _xval = tf.convert_to_tensor(xval, dtype=getattr(tf, db.dtype))
159                _yval = tf.convert_to_tensor(yval, dtype=getattr(tf, db.dtype))
160                trai = tf.data.Dataset.from_tensor_slices((_xtrain, _ytrain)).batch(db.batch_size).with_options(options)
161                val = tf.data.Dataset.from_tensor_slices((_xval, _yval)).batch(db.batch_size).with_options(options)
162
163                history = self.model.fit(trai, batch_size=db.batch_size, epochs=epochs,
164                                         validation_data=val,
165                                         callbacks=[
166                                             fit_callback,
167                                             tf.keras.callbacks.TensorBoard(log_dir=f'{__tensorboard_logs__}/'
168                                                                                    f'{self.name}'),
169                                             tf.keras.callbacks.EarlyStopping(patience=10),
170                                             tf.keras.callbacks.ModelCheckpoint(filepath=f'{__keras_checkpoint__}'
171                                                                                         f'{self.name}.h5')
172                                         ])
173                __history__ = BaseNetResults(history.history['loss'], history.history['val_loss'])
174
175        except Exception as ex:
176            if self._verbose:
177                logging.error(f'BaseNetModel: Cannot train the model, it raised an exception: {ex}.')
178        finally:
179            return __history__

181    def predict(self, x, scale: float = 1.0, th: (None, float) = 0.5, expand_dims: bool = False) -> tf.Tensor:
182        """
183        This function predicts with the current model an output from the input 'x', divided by the 'scale' and converted
184        to a binary matrix with a custom threshold 'th'.
185        :param x: Input np.array o tf.Tensor.
186        :param scale: Divides by a custom scale (default: 255.0 -> 8bit images).
187        :param th: Custom output threshold (default: 0.5 -> mid-range predictions). Set up to 'None' to see
188        the real output.
189        :param expand_dims: Expands the dimension of the tensor.
190        :return: The prediction output of the model.
191        """
192        _x_ = tf.convert_to_tensor(np.array(x).astype("float32") / scale)
193        if expand_dims:
194            __x = tf.expand_dims(_x_, axis=-1)
195        else:
196            __x = _x_
197        _y_ = self.model.predict(__x)
198        if th is not None:
199            __y__ = self._threshold(_y_, th)
200        else:
201            __y__ = _y_
202        return __y__

204    def evaluate(self, ndb, metric, th: (None, float) = 0.5) -> (tf.Tensor, None):
205        """
206        This method evaluates the test dataset of a BaseNetDatabase.
207        :param ndb: Number of the database to be tested.
208        :param metric: A metric function.
209        :param th: The threshold of the prediction, if not None.
210        :return: The evaluated metric.
211        """
212        if ndb < len(self.breech):
213            db = self.breech[ndb]
214        else:
215            if self._verbose:
216                logging.warning('BaseNetModel: Cannot load the BaseNetDatabase to evaluate, '
217                                'the index of the database does not exist.')
218            return None
219        xtest = tf.convert_to_tensor(db.xtest, dtype=getattr(tf, db.dtype))
220        ytest = tf.convert_to_tensor(db.ytest, dtype=getattr(tf, db.dtype))
221        _output_ = self.predict(xtest, th=th, scale=1.0)
222        result = metric(_output_, ytest)
223        return result

347    def call(self, *args, **kwargs):
348        """
349        When calling the class BaseNetModel, you can merge two different models. So the model1(model2) will merge the
350        model2 into the model1. By default, all the model parameters and options are inherited from the model1.
351
352        :param args: Incoming model; BaseNetModel or tf.keras.model.
353        :param kwargs: {'name': the model name; inherits the model1 name by default,
354        'parallel': True for separate inputs, False to be sequential; False by default,
355        'options': compile options; inherits the model1 compile options by default}
356        :return: A bypass of the current object.
357        """
358        if len(args) != 1:
359            if self._verbose:
360                logging.warning('BaseNetModel: The number of arguments to the callable class must be 1.')
361            return self
362
363        input_model = args[0]
364        if not isinstance(input_model, (keras.models.Model, BaseNetModel)):
365            if self._verbose:
366                logging.warning(f'BaseNetModel: The input model must be a BaseNetModel or a tf.keras.model not a '
367                                f'{type(input_model)}.')
368            return self
369        else:
370            if isinstance(input_model, BaseNetModel):
371                keras_model = input_model.model
372            else:
373                keras_model = input_model
374
375        the_model_is_parallel = False
376        the_model_is_under_the_current_model = True
377        name = self.name
378        options = self.compiler.compile_options
379        for key, item in kwargs.items():
380            if key == 'parallel':
381                the_model_is_parallel = item
382            elif key == 'top':
383                the_model_is_under_the_current_model = item
384            elif key == 'name':
385                name = item
386            elif key == 'options':
387                options = item
388
389        if not the_model_is_parallel:
390            if the_model_is_under_the_current_model:
391                result_model_layers = []
392                result_model_layers.extend(self.model.layers)
393                result_model_layers.extend(keras_model.layers[1:])
394            else:
395                result_model_layers = []
396                result_model_layers.extend(keras_model.layers)
397                result_model_layers.extend(self.model.layers[1:])
398
399            _in = result_model_layers[0]
400            mids = _in
401            for layer in result_model_layers:
402                mids = layer(mids)
403            _out = mids
404            result_model = keras.models.Model(inputs=[_in], outputs=[_out], name=name)
405
406        else:
407            _in0 = keras.Input(shape=self.model.input_shape)
408            _in1 = keras.Input(shape=keras_model.input_shape)
409            _out = keras.layers.concatenate([_in0, _in1])
410            result_model = keras.models.Model(inputs=[(_in0, _in1)], outputs=[_out], name=name)
411
412        result_model.compile(options)
413        self.model = result_model
414        return self

430    def _build(self) -> keras.Model:
431        _scope = self._get_scope(self.compiler.devices)
432
433        with _scope:
434            # Add the input of the model.
435            _inp = keras.Input(shape=self.compiler.io_shape[0])
436            _inp._name = 'compiled-model-keras'
437            _lastlay = _inp
438
439            last_master = None
440            pipeline_opened = False
441            towers = []
442
443            for layer in self.compiler.layers:
444                for layer_type, (layer_shape, layer_args) in layer.items():
445                    # Core layers:
446                    if layer_type == 'open_pipeline':
447                        if not pipeline_opened:
448                            last_master = _lastlay
449                        else:
450                            towers.append(_lastlay)
451                            _lastlay = last_master
452                        pipeline_opened = True
453                    elif layer_type == 'close_pipeline':
454                        pipeline_opened = False
455                        towers.append(_lastlay)
456                        _lastlay = keras.layers.concatenate(towers, **layer_args)
457                        towers = []
458                    else:
459                        if layer_type in KERAS_LIST_LAYERS and layer_type not in PREBUILT_LAYERS:
460                            this_lay = getattr(keras.layers, layer_type)(*layer_shape, **layer_args)
461                        elif layer_type in KERAS_LIST_LAYERS:
462                            this_lay = getattr(Subkeras, layer_type)(*layer_shape, **layer_args)
463                        else:
464                            importmodel = keras.models.load_model(f'{layer_type}.h5')
465                            importmodel._name = layer_type
466                            this_lay = importmodel
467
468                        _lastlay = this_lay(_lastlay)
469
470            # Add the output of the model.
471            out = keras.layers.Dense(self.compiler.io_shape[1], activation="sigmoid", name='output')(_lastlay)
472            _compile = copy.copy(self.compiler.compile_options)
473
474            if _compile['loss'] in PREBUILT_LOSSES:
475                _compile['loss'] = getattr(Sublosses, self.compiler.compile_options['loss'])
476
477            model = keras.Model(_inp, out, name=self.name)
478            model.compile(**_compile)
479        return model

 39    The BaseNetModel class provides a wrapper for the tf.keras.model API with easier use. When initialized,
 40    it initializes a breech of databases in its attribute 'breech'. If we provide a compiler, the model will be
 41    built from the compiler; however, if we provide a tf.keras.model, the compiler is ignored and the model is built
 42    from the provided tf.keras.model.
 43
 44    To add a database to the model, we can use the method BaseNetModel.add_database() that takes a BaseNetDatabase as
 45    input.
 46
 47    The class contains load and save methods to store the compilers (.cpl files) and models (.h5 files) in the same
 48    directory.
 49
 50    We also provide a BaseNetModel.fit() method that can create a separate process for training. The original framework
 51    does not include this feature:
 52        *   The BaseNetModel.fit() method takes as input the index of the loaded database via
 53        BaseNetModel.add_database() method and takes the train and validation subsets to fit the model.
 54        *   If the training process should not block the main process, the parameters 'avoid_lock' must be set to True,
 55        in that case, another process will take over the fitting tf.keras.model.fit() method and the information will
 56        be updated in the return class: BaseNetResults.
 57        *   In case we avoid the main process to be locked with the 'avoid_lock' feature, we will need to recover the
 58        tf.keras.model with the BaseNetModel.recover() method once the training is finished (check
 59        BaseNetResults.is_training).

 62    test subset.

350        When calling the class BaseNetModel, you can merge two different models. So the model1(model2) will merge the
351        model2 into the model1. By default, all the model parameters and options are inherited from the model1.

360            if self._verbose:
361                logging.warning('BaseNetModel: The number of arguments to the callable class must be 1.')

366            if self._verbose:
367                logging.warning(f'BaseNetModel: The input model must be a BaseNetModel or a tf.keras.model not a '
368                                f'{type(input_model)}.')
369            return self
370        else:
371            if isinstance(input_model, BaseNetModel):
372                keras_model = input_model.model
373            else:
374                keras_model = input_model
375
376        the_model_is_parallel = False
377        the_model_is_under_the_current_model = True
378        name = self.name
379        options = self.compiler.compile_options
380        for key, item in kwargs.items():
381            if key == 'parallel':
382                the_model_is_parallel = item
383            elif key == 'top':
384                the_model_is_under_the_current_model = item
385            elif key == 'name':

387            elif key == 'options':

431    def _build(self) -> keras.Model:
432        _scope = self._get_scope(self.compiler.devices)
433
434        with _scope:
435            # Add the input of the model.
436            _inp = keras.Input(shape=self.compiler.io_shape[0])
437            _inp._name = 'compiled-model-keras'
438            _lastlay = _inp
439
440            last_master = None
441            pipeline_opened = False
442            towers = []
443
444            for layer in self.compiler.layers:
445                for layer_type, (layer_shape, layer_args) in layer.items():
446                    # Core layers:
447                    if layer_type == 'open_pipeline':
448                        if not pipeline_opened:
449                            last_master = _lastlay
450                        else:
451                            towers.append(_lastlay)
452                            _lastlay = last_master
453                        pipeline_opened = True
454                    elif layer_type == 'close_pipeline':
455                        pipeline_opened = False
456                        towers.append(_lastlay)
457                        _lastlay = keras.layers.concatenate(towers, **layer_args)
458                        towers = []
459                    else:
460                        if layer_type in KERAS_LIST_LAYERS and layer_type not in PREBUILT_LAYERS:
461                            this_lay = getattr(keras.layers, layer_type)(*layer_shape, **layer_args)
462                        elif layer_type in KERAS_LIST_LAYERS:
463                            this_lay = getattr(Subkeras, layer_type)(*layer_shape, **layer_args)
464                        else:
465                            importmodel = keras.models.load_model(f'{layer_type}.h5')
466                            importmodel._name = layer_type
467                            this_lay = importmodel
468
469                        _lastlay = this_lay(_lastlay)
470
471            # Add the output of the model.
472            out = keras.layers.Dense(self.compiler.io_shape[1], activation="sigmoid", name='output')(_lastlay)
473            _compile = copy.copy(self.compiler.compile_options)
474
475            if _compile['loss'] in PREBUILT_LOSSES:
476                _compile['loss'] = getattr(Sublosses, self.compiler.compile_options['loss'])
477
478            model = keras.Model(_inp, out, name=self.name)
479            model.compile(**_compile)
480        return model

The BaseNetModel class provides a wrapper for the tf.keras.model API with easier use. When initialized, it initializes a breech of databases in its attribute 'breech'. If we provide a compiler, the model will be built from the compiler; however, if we provide a tf.keras.model, the compiler is ignored and the model is built from the provided tf.keras.model.

To add a database to the model, we can use the method BaseNetModel.add_database() that takes a BaseNetDatabase as input.

The class contains load and save methods to store the compilers (.cpl files) and models (.h5 files) in the same directory.

We also provide a BaseNetModel.fit() method that can create a separate process for training. The original framework does not include this feature: * The BaseNetModel.fit() method takes as input the index of the loaded database via BaseNetModel.add_database() method and takes the train and validation subsets to fit the model. * If the training process should not block the main process, the parameters 'avoid_lock' must be set to True, in that case, another process will take over the fitting tf.keras.model.fit() method and the information will be updated in the return class: BaseNetResults. * In case we avoid the main process to be locked with the 'avoid_lock' feature, we will need to recover the tf.keras.model with the BaseNetModel.recover() method once the training is finished (check BaseNetResults.is_training).

We can also evaluate the performance of the database with the BaseNetModel.evaluate() method, that makes use of the test subset.

We can also predict the output of a certain input with the BaseNetModel.predict() method.

We can also visualize the model with the BaseNetModel.print() method in a PNG image.
